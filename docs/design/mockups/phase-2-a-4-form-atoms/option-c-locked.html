Paper-card chrome, uniform.

Input · Select · Textarea · Label all share one state machine. Three ink weights signal fill progression (idle → filled → focused). Paper press idiom carries focus. Jersey-deep stays out of the field — it's reserved for content elsewhere on Track B. Alert color + alert-soft shadow uniquely mark errors.
