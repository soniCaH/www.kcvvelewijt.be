↑ FOR

Strip format reads as fanzine artefact — multiple crops feel like a personality reveal, not a corporate headshot. Jersey silhouette renders the team colour at hero scale without a single glow or rim-light. TicketStub + MonoLabel caption reuse production primitives verbatim. Italic serif number anchors the strip and gives an editorial weight you can't get with a sticker.

↓ AGAINST

Four photo crops require either four real psdImages or a clever multi-crop pipeline; for most players we only have one rectangular portrait. Mitigation: drive all four frames from the same image with different background-position + background-size values. Strip is taller than a single portrait — vertical real estate cost on grid pages.

↑ FOR

Fallback stays inside the same frame — no separate empty-state primitive. Monogram on jersey-deep keeps team identity even when the player has zero usable photos. Tight crop variant proves the same component absorbs different source aspect ratios without breaking layout.

↓ AGAINST

Fallback monogram needs a deterministic colour mapping (we used jersey-deep here); without that rule, fallbacks drift to whatever the implementer feels. Tight-crop "feels" identical to default unless side-by-side — discoverable only on PSD-imported portraits where the source crop is unusual.

★ 03 · JERSEYSHIRT — 4-up: home stripes · away · retro · training · letter-overlay support

↑ FOR

Single SVG-free clip-path silhouette renders all four variants — fill colour and overlay are the only deltas. Letter overlay slot supports age-group labels and number reuse without new typography. Cards inherit the canonical press-down hover (try clicking).

↓ AGAINST

Block-colour silhouette reads as flat by design — fans expecting a kit photo on a "shop the kit" page will find this stylised. Mitigation: this is a chrome figure, not a product image; the actual webshop kit photos remain photographic.

★ 04 · ENDMARK — centred typographic asterisk-cluster · jersey-green flank

Het einde van het gesprek.

<EndMark> closes long-form articles, interviews and Q&A pages. A pure typographic glyph cluster — three asterisks in Playfair 900 with subtle vertical jitter — flanked by the phrase * Einde gesprek * in italic display serif. Jersey-green ink on cream. No icon SVG, no horizontal rule.

* Einde gesprek *
CLUSTER three-asterisk variant for chapter close (no phrase)
***

↑ FOR

Pure type — zero SVG, zero new tokens. Asterisk is a universally-readable end-of-article glyph, and Playfair 900 gives it editorial weight without invoking icons. Jersey-green ink ties the figure to team identity in a single property.

↓ AGAINST

Three asterisks alone can read as Markdown debris if a reader skims. The phrased variant fixes this but adds Dutch copy (Einde gesprek) — needs a localisation slot if the site ever ships a non-Dutch locale. For non-conversational articles the phrasing feels narrow; provide a phrase prop.

★ 05 · QASECTIONDIVIDER — horizontal taped paper strip · title slot · pull-quote slot

Tussen de aktes.

Long Q&A pieces split into acts. <QASectionDivider> is a full-width horizontal taped strip with two slots: an italic display title left, a pull-quote centre, and a typographic flourish right. Tape strips at both upper corners visually nail the divider to the page.

↑ FOR

Taped strip reuses the exact tape vocabulary already on TapedCard — readers see continuity, not a new chrome atom. Three slots (title / quote / flourish) give the long-read a real chapter break instead of a horizontal rule. Pull-quote slot gives editors something to do that isn't markdown.

↓ AGAINST

Three-column inner grid collapses awkwardly on narrow screens — needs a stacked mobile layout (title on top, quote middle, flourish bottom). Pull-quote is optional editorial work; not every Q&A will have one, so the figure must look balanced when only title + flourish are present.
