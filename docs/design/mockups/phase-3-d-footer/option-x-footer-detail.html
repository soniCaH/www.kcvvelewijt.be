★ 03 · RENDER RULE — footer renders on every page (root layout slot)

Wanneer rendert de footer?

Unlike the MatchStrip (Checkpoint C Q5 G3 — landing-only), the footer is a colofon anchor: copyright, address, legal links, and cookie controls must reach every page including detail templates and hero-less utility pages.

Page type Footer render Notes
Homepage / Render SponsorsSection sits above footer (separate); footer below
Section indexes (/nieuws, etc.) Render Footer below content + MatchStrip (when present)
Article detail (any articleType) Render Footer below body — chrome-free article zone above; footer is colofon
Single team / player / match detail Render Same colofon role
Hero-less utility (/zoeken, error pages) Render Cookie + privacy controls reach error pages too
