↑ FOR

Lightest of the four directions. The stamp is the entire retro anchor — typography carries the brand, not chrome. Tint per variant is unmistakable but never loud. Sharp corners + mono caps stamp = "printed notice".

↓ AGAINST

Without a paper-card body, the alert can blur into surrounding cream content if the parent layout is also untextured. The stamp must do all the lifting — one minor visual element away from feeling like a generic flash banner.

Title hierarchy is clean — bold Inter against body Inter, separated by a hairline dashed rule that nods to match-card__footer. Dismiss X is unobtrusive but reachable; no chrome around it competes with the stamp.

★ 04 · TRADEOFFS — why this direction wins · what it sacrifices

Wegen en weegschalen.

↑ FOR

Most restrained of the four candidates. Leans on typography (mono caps stamp) rather than chrome (paper-card body) to carry the retro signal — the "less is more" reading of Direction D. Composes with FilterTabs / BrandedTabs vocabulary because it shares the fonts and token palette, not because it copies their border-2 ink shell. The hand-rotated stamp is unmistakably ours; readers spot the typographic glyph + mono caps before they read the message. Sharp corners, ink + cream + retro-pigment palette, no neon, no radial glow. Per-variant tint is loud enough to scan but never feels like a Material toast.

↓ AGAINST

Chrome lives in one place — if the stamp is ever suppressed (e.g. an extra-tight inline context where 56px is still too wide), the alert collapses to a tinted strip with a body and no retro signal at all. There is no visual fallback. Compared to the paper-card directions, the alert sits flatter on the page — fine in a section panel, possibly too quiet on a busy editorial homepage where every other module has shadow-paper-sm chrome. The dashed paper-edge rule helps the title/body split, but it remains optional — without it, hierarchy depends purely on font weight, which is the thinnest of safety nets.

★ NOTES — token map for the alert atom

Token map

SURFACE / TINT

  • Success bg: 10% jersey on cream (#e6f4e2)
  • Warning bg: --color-warning-soft
  • Error bg: --color-alert-soft
  • Alert outer border: 1px solid var(--color-paper-edge) (subtle)
  • Title + body: --color-ink
  • Dashed rule: 1px dashed var(--color-paper-edge)

STAMP

  • Stamp border: 2px solid var(--color-ink) — the only heavy ink
  • Stamp accent (icon + label): jersey-deep / warning / alert
  • Stamp size: 64px square (default), 56px (inline)
  • Stamp rotation: --rotate-stamp-a..c cycle via nth-child(3n+1..3)
  • Stamp label: --font-mono 9px (default) / 8px (inline), 700 weight, uppercase, two lines
  • Stamp ink texture: low-opacity radial dots, easy on the eye
  • Sharp corners: no border-radius anywhere
