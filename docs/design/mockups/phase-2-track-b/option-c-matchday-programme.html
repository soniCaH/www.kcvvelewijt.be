★ Phase 2 · Track B · Direction C

Matchday
programme.

Een hybride richting tussen Paper en Mono. We lenen het vocabulair van een gedrukt matchblad: getelde badges, genummerde secties, sjaal-strepen voor laad-states en geperforeerde ticket-stubs voor actieve tabbladen. Chrome blijft grotendeels plat — rotatie wordt zeldzaam ingezet (alleen voor het actieve ticket-tabblad). Data wordt grafisch gecodeerd zodat fans het herkennen vóór ze het lezen.

− Tegen

Surface-extensie: subLabel?: string moet als optionele prop op BrandedTabs verschijnen. Mobile-portrait: 4 tickets in een rij wordt te smal — vereist horizontaal-scroll behavior of vertical stack.

Direction-level trade-offs

+ Argument FOR Direction C

Voetbalfans herkennen programme-vocabulair direct: counts, genummerde secties, sjaal-strepen, ticket-stubs zijn cultureel beladen. Dit is een toegankelijke hybride die data graphisch codeert zonder Direction-A's over-rotation, en zonder Direction-B's restraint te zwaar in te zetten.

− Argument AGAINST Direction C

Meer visuele complexiteit dan restraint. Risico: chrome kan concurreren met editorial primitives (interview-spreads, photo-essays) als ticket-stubs en stamp-chips overal opduiken. Discipline rond plaatsing is essentieel — programma-elementen mogen alleen op dataschermen verschijnen, niet op redactionele pagina's.

Atom prop-surface impact

  • Spinner — geen surface-wijziging. variant: "logo" blijft beschikbaar als fallback voor <80px.
  • BrandedTabs — sub-variant 01 (sliced) leidt tab-nummer af van index, geen nieuwe prop. Sub-variant 03 (ticket) vereist subLabel?: string per tab. Voorkeur voor 03 → flag dit als kleine surface-extensie.
  • FilterTabs — past binnen huidige surface; count is al gemodelleerd. Geen wijziging.
  • HorizontalSlider — variant A (header counter+arrows) vereist optionele total?: number + current?: number; en een title-row layout-shift. Variant B en C vereisen geen nieuwe props.
  • ScrollArrowButton — variant A (chunky) vraagt size?: "sm" | "md" | "lg" — bestaande styling kan worden behouden voor sm; lg wordt de 56×56 chunky.
