How variety is created (your "don't all look the same")

  • Register rhythm — cream/dark alternates down a match (above: A·B·B·A·A·A·B), so the feed never reads as one repeated card.
  • Per-event shape — fixture, goal, scoreboard, sober-goal, disciplinary and result all compose differently; same-register cards (4·5·6) still look distinct.
  • Photo vs no-photo goal — goal #2 (taped newsprint photo) vs goal #3 (giant ghost shirt-number hero) are two different goal layouts the system picks between automatically.
  • Subtle per-card tape angle + rotation from the existing --rotate-tape-* pool keeps repeated goals from being pixel-identical.

Guardrails honoured

  • No overlap / text fits — flex column (top / hero / footer); hero text is size-capped and wraps inside the frame. Implementation keeps a fixed vertical budget per zone.
  • Square corners · ink borders + hard offset shadow · paper-grain · Freight + Montserrat + IBM Plex Mono (Quasimoda retired) · photos in colour newsprint.
  • No magazine/edition chrome — only club identity (crest · @kcvvelewijt · kcvvelewijt.be).
