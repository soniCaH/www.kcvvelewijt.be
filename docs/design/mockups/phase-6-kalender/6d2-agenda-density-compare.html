Phase 6.D · 6d2 — Agenda view: dense-day mitigation

The 6d1 lock ships BOTH views (grid default + agenda tab). The agenda's one weakness is the dense Saturday — 10 near-identical match rows that bury the occasional event. Same Zaterdag 12 september (10 wedstrijden + Spaghetti-avond) rendered three ways. Which keeps the agenda readable AND never buries events? (Grid view already handles density; this is purely the agenda tab.)

(i) Labelled wall — show everything
All rows, always. Day-count header sets expectation; events get a tinted row so they stand out. Zero interaction — everything is on the page.

(ii) Collapse matches ▸ — events always open
Matches fold into one tappable group per day ("10 wedstrijden ▸", collapsed by default); events render in full beneath, never buried. One tap expands the matches. Honours the by-type filter (selecting Wedstrijden auto-expands).
