Phase 6.D · 6d5 — iCal subscribe panel (“volg je ploeg”)

The subscribe panel is a matches-only “follow your team” webcal feed (orthogonal to the by-type filter; 6d lock §iCal). Function is fixed: pick team(s) → optional thuis/uit → get a webcal:// link (copy or QR) to add to a calendar app. No design was drilled at the gate — only “reskin the chrome to paper/ink.” Three treatments of the SAME fields. Harness chrome (dark bars) is not part of the design.

(i) Clean stack — utilitarian
The reskinned current IA, tidied. Team toggle-chips (multi-select, ink fill = selected), a thuis/uit segmented control, then the link + copy/QR. Everything visible at once; no steps. Neutral ink chips (not card-red) so they don’t read as the Wedstrijden filter.
