Two row shapes for the team-schedule list.

Sole consumer is <TeamSchedule> on /ploegen/[slug] (light theme). Each variant rendered as a vertical stack of 5 match rows — recent results for KCVV with mixed W / G / L outcomes — so the per-row visual at list scale reads.

Variant A

Mini-teaser row.

Same shape as <MatchTeaser> from d6 (A2-italic stub + body), scaled down to row height. Date stamp on the left, teams + score in the body, result pill at the row end. Strongest visual coherence with the rest of the page.

Variant B

Result-anchored row.

Date column → home → score → away → result pill. Score is the visual anchor (display-big 22px). Result reinforced by a subtle row-bg tint (win = cream, draw = cream-soft, loss = cream-deep). Reads as a results table.
