F — Logo + left/right split (B + D)

Spatial split · logo replaces short code

Pro: the row immediately tells you which side owns the event without decoding. Logo carries recognition; mono code is dropped (less clutter). Con: halved name columns; long names truncate earlier. Mobile collapse will need to flatten back to a single column. 7-column grid.
