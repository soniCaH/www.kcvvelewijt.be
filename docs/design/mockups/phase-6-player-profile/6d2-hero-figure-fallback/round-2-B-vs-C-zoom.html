The deciding question

For each missing-photo profile, do you want the page to whisper "this is player #17" (Variant B — the number on the shirt is the identity), or "this is a player at the club" (Variant C — generic figure, identity carried by the #8 / name in the left column only)?

Variant B keeps each profile feeling individual at the cost of a slight vocabulary delta (JerseyShirt gains a number prop). Variant C is zero-vocabulary-delta but produces an identical illustration across all fallback profiles — a viewer paging through the squad would see the same figure on profiles 3, 11, and 19.

Both variants share the same <TapedFigure> frame, the same tape strip, the same jersey-deep + ink + stripes vocabulary. The figure-vs-shirt is the only difference.
