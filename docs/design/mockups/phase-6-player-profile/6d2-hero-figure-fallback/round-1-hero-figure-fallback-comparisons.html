Hero photo fallback treatment

Brief locks the present hero image as <TapedFigure aspect="portrait-3-4"> with newsprint filter + paper grain. What's open: what renders when player.psdImage is missing — roughly 10% of any 24-man squad per [[feedback_subject_photo_fallback]].

Each column below shows the locked present state on top (identical across variants) and the proposed fallback below. The variants only diverge in the fallback row. Variant D drops the photo column entirely — the 4th drop-section variant per the 6.d0 lock.

Monogram.

Missing image → display the player's initials in oversized italic Black inside the same taped frame. Honors paper aesthetic; quiet; zero net-new vocabulary. Reads as a placeholder, not a portrait.
