Player-name typography rhythm

Brief Q1 — header identity. The hero stacks two lines: first name + last name. Each variant below shows the same Maxim Breugelmans hero (number, photo, meta) with only the name treatment changed. Shown at desktop hero scale and a 260px mobile strip — name rhythm reads differently as it shrinks.

Brief Q1's options A/B/C kept. Variant D opens new ground: a "stage-name" treatment that promotes the last name and demotes the first name to a mono kicker. Honors the "drop / radically reframe" slot from the owner direction on the 6.d0 lock.

Variant A

Uniform Black.

Both names share the same display italic Black weight. Current <EditorialHeading> default — no prop extension needed. Quiet, refined, single voice.
