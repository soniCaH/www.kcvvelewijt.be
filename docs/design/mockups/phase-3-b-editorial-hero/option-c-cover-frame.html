PHASE 3 · CHECKPOINT B · EDITORIAL HERO · DIRECTION C ★ COVER FRAME 2026-05-03

Een kaft rond het verhaal.

Direction C reads the hero as a magazine cover. A 2px ink-bordered rectangle with a 6px hard offset shadow contains a layered composition: mono kicker top, editorial heading + dek + CTA bottom-left, an artefact on the right that deliberately overlaps the frame edge to give the page a tactile object-on-paper read. Variant differs only in artefact + kicker — heading, dek, and CTA structure are invariant. One component, four locked variants (originally specced as seven). No glows, no rim-lights, sharp corners, press-down hover throughout.

★ 10 · DIRECTION-LEVEL TRADEOFFS — summary across all four locked variants

Waarom kaft.

Direction C reads the EditorialHero as a contained editorial object, not a broadsheet. Concentrated, framed, more "magazine cover" than "front page". One frame, four locked variants, every artefact composed from existing primitives.

↑ FOR

Maximum chrome reuse — the frame is just border + shadow + cream, identical vocabulary to Alert, BrandedTabs, TapedCard. Artefacts breaking the frame edge give the cover tactile depth without animation or glow. Variant differentiation is artefact + kicker only — heading / dek / CTA are constant, so editorial workflow stays simple. Press-down hover canon applied throughout. Sharp corners, no rounded edges. Stenciletta off-limits respected. Photo-first for player figures per the rectangular-default rule.

↓ AGAINST

The "object on paper" reading depends on artefacts overflowing the frame edge — at narrow breakpoints (≤880px) we lose that effect and the variant collapses to a stacked panel. Frame + artefact creates a 2-layer paper stack which on a stacked feed (NewsGrid) risks reading as "more chrome on top of chrome". Direction asks the cover to do most of the editorial work, so the announcement variant leans heavier on typography than imagery — the kind of risk that originally motivated a separate generic variant (now removed — see scope correction admonition above).

★ NOTES — token map, structure decisions, production handoff

Resolved

  • Cover frame = 2px solid var(--color-ink) + --shadow-paper-md (6×6 ink) + sharp corners + cream / cream-soft / ink / jersey-deep surface
  • Variant differentiation is artefact + kicker only — heading / dek / CTA structure is constant across all four locked variants
  • Artefact area sits in the right grid column and may overflow the frame's right or bottom edge by up to 56px (transfer / event) — the parent cover-stage reserves margin for that overflow. (originally also listed match-preview / player; both removed from Phase 3 B scope per the admonition above — match-preview defers to issue #1470, player to Phase 6)
  • All four locked variants compose existing primitives only (TapedCard, TicketStub, MonoLabel, EditorialHeading, ClippedCard, StampBadge)
  • Generic variant introduces a low-opacity --font-display ornament treatment — composed from existing tokens, not a new primitive [REMOVED — generic variant out of Phase 3 B scope, see admonition above]
  • Press-down hover canon: hover:translate(4px, 4px) + hover:shadow: none + transition-all 300ms on every TapedCard, ticket-stub-bearing primitive, and CTA
  • No border-radius anywhere, no radial glows, no rim-lights — paper-flat shadows only
  • Subject photos rectangular default per the photo-fallback rule (PlayerFigure renders player.psdImage as 3:4 bounded box)
  • Mobile collapse below 880px: grid drops to one column, artefact moves under body, no more frame-edge overflow
  • Stenciletta is not used

Token map

SURFACE / TINT

  • Frame surface (default): --color-cream
  • Frame surface (alt): --color-cream-soft
  • Frame surface (inverted): --color-ink with cream text
  • Frame surface (jersey): --color-jersey-deep with cream text
  • Heading colour: --color-ink (cream on inverted / jersey)
  • Heading emphasis colour: --color-jersey-deep (jersey-bright on inverted / jersey)
  • Dek colour: --color-ink-soft (cream @ 86% on inverted / jersey)
  • Kicker rule: --color-ink @ 40% (cream @ 100% on inverted / jersey)

STRUCTURE / DEPTH

  • Frame border: 2px solid var(--color-ink)
  • Frame shadow: --shadow-paper-md (6×6 ink)
  • TapedCard shadow: --shadow-paper-md (6×6 ink)
  • TicketStub shadow: --shadow-paper-sm (4×4 ink)
  • Press-down hover: translate(4px, 4px) + box-shadow: none + transition: all 300ms ease
  • Sharp corners: no border-radius anywhere
  • Artefact rotation range: -3.1deg (rot-b) to +2.4deg (rot-a)
  • Artefact overflow allowance: up to 56px outside frame on right edge, 22px below frame
