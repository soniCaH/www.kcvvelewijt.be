PHASE 3 · CHECKPOINT B · EDITORIALHERO · DIRECTION A ★ ASYMMETRIC BROADSHEET 2026-05-03

Een broadsheet hero.

One <EditorialHero> with seven variants. The headline / dek / CTA structure is invariant — only the right-column artefact and the kicker label set switch per variant. Layout is a 60 / 40 left-right split with a thin ink rule between columns and generous editorial white space. Heavy Playfair display type on the left, paper-shadow TapedCard artefact on the right with a slight rotation. No radial glows, no rim-lights — paper-flat shadows only. Press-down hover canon (translate(4px, 4px) · shadow → none · 300ms) on every interactive primitive.

★ 10 · DIRECTION TRADEOFFS — summary across all seven variants

Waarom een broadsheet.

Direction A reads as a Saturday paper front-page: a heavy headline on the left, a tilted artefact on the right, a thin rule between them. The 60/40 split is the most common newspaper hero ratio — enough room for a 5–6 word italic headline without the artefact shrinking into a thumbnail. Lowest typographic risk; tightest editorial reading.

↑ FOR

Maximises the editorial language the redesign was hired to speak — Playfair display in italic, mono-caps kickers, paper shadows, jersey-deep accent. Every artefact composes from already-shipped primitives (TapedCard, StampBadge, PlayerFigure, TicketStub, MonoLabel, EditorialHeading) — no new shapes shipped by this component. The 60/40 split scales cleanly to one column on mobile by stacking artefact-above-text, which matches the cross-template default. Press-down hover canon applies uniformly to buttons + cards. Single component, seven variants, one type-discriminated prop — cheaper to ship, cheaper to maintain.

↓ AGAINST

Heavy on the headline — variants where the article title is short (announcement: "De kantine blijft open.") leave a lot of empty cream in the left column, while variants where the headline runs long (interview: 5+ words) wrap to three lines and pressure the CTA placement. Two-column tension is universally specified across hero / interlude / quote — risk of "every section looks like a hero" if the rest of the page chrome doesn't deliberately break the rhythm. The poster variant's ink-bg artefact and the announcement's StampBadge circular shape introduce more variance inside the right column than the other five variants combined — the discriminated union has internal tension between "geometric primitive variants" and "editorial poster variants".

★ NOTES — production handoff for EditorialHero

Resolved

  • Layout grid: grid-template-columns: 60% 40% with a 1px ink rule via ::before; collapses to single column at ≤ 900px
  • Left column is invariant: MonoLabelRow + EditorialHeading size="display-xl" + dek paragraph + button row + byline
  • Right column varies by variant prop only; every artefact wraps in TapedCard with rotation modifier (-2.4° / +2.1° / -1.4°)
  • Headline italic-emphasis span uses --color-jersey-deep — same convention as EditorialHeading emphasis
  • Press-down hover canon uniform across buttons + TapedCard: transition: all 300ms; on hover { transform: translate(4px, 4px); box-shadow: none; }
  • Player variant uses rectangular psdImage crop (saved-memory: subject photos default to rectangular)
  • StampBadge announcement variant requires headline copy ≤ 2 words for inner double-rule fit
  • No tape on the announcement variant (TapedCard tape collides with rotated StampBadge); use tape="alt" position offset
  • Mobile stacking: artefact moves above the headline (cross-template default); vertical ink rule hidden; padding tightens to 32px

Token map

SURFACE / TINT

  • Hero surface: --color-cream
  • Hero border: 2px solid var(--color-ink)
  • Hero shadow: --shadow-paper-md (6×6 ink)
  • Vertical rule: 1px solid var(--color-ink) at opacity: 0.55
  • TapedCard surface: --color-cream
  • TapedCard shadow: --shadow-paper-lg (10×10 ink)
  • Headline emphasis: --color-jersey-deep
  • Poster artefact bg: --color-ink w/ --color-jersey accent
  • Player photo source: rectangular player.psdImage

STRUCTURE / DEPTH

  • Grid columns: 60% 40% desktop
  • Grid columns: 1fr at ≤ 900px (artefact stacks above)
  • Headline scale: clamp(56px, 6.4vw, 96px)
  • Dek scale: 18px desktop / 15px mobile
  • Press-down hover: transform: translate(4px, 4px); box-shadow: none;
  • Transition: all 300ms ease
  • TapedCard rotations: -2.4° / +2.1° / -1.4°
  • StampBadge rotation: -9° (counter-clockwise)
  • Sharp corners: no border-radius on hero, buttons, taped-card, ticket-stub
