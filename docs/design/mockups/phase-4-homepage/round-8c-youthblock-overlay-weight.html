Phase 4 · Round 8c · #1700 · LOCKED 2026-05-08

YouthBlock — overlay weight & pattern exploration

Locked 2026-05-08: Q1 → 8c.B (#133d28 third stop) + Q2 → 8c.D with dots softened to 0.05 alpha. Three tokens ship in #1697: --color-jersey-deep-dark (atomic), --gradient-jersey-deep-overlay (composed baseline 88/65/92), --pattern-halftone-dots (5% cream dots, 8×8 grid, screen blend). 8c.A/8c.C/8c.E retained for the record.

Implementation feedback on round-8b N.2 (#1675): the photo reads through too much against the live brand --color-jersey-deep: #008755. The round-8b mockup used #1f5f3f (desaturated forest) which never landed in live tokens. This round answers two questions:

Q1 — palette: brand-consistent #005a39 (8c.A) vs. desaturated mockup #133d28 (8c.B) for the third gradient stop / new --color-jersey-deep-dark token (#1697).

Q2 — overlay strategy: baseline 88/65/92 alphas (8c.A), heavier 95/82/97 (8c.C), or baseline + retro pattern overlay (8c.D dot halftone, 8c.E barber-pole stripes).

Out of scope
Diagonal section seams. The diagonal-to-StripedSeam migration is tracked at #1701 — every variant below renders YouthBlock as a standalone surface so the comparison stays clean.
8c.A · Q1 — brand-consistent palette

Baseline alphas, #005a39 third stop

Same 88/65/92 alpha steps as round-8b N.2, but the third stop lands on a brand-darkened green (a darker shade of live #008755). Photo bleeds through; gradient stays tonally consistent with existing jersey-deep surfaces site-wide.

Pro Tokens consistent with FeaturedEventBand, StampBadge, etc.
Con Lighter / brighter feel than the round-8b mockup intent.

8c.C · Q2 — heavier overlay

Photo as ghost texture (95/82/97 alphas)

Crank the overlay so the photo barely surfaces — visible only as a subtle texture in the lighter regions. Removes any "stock photo" read; the photo becomes paper noise rather than imagery. Owner feedback on #1675 leaned toward heavier overlay; this answers it.

Pro Photo never competes with text; consistent with FeaturedEventBand's ink-dense feel.
Con Why have the photo at all? May warrant dropping it (→ #730 photo curation backlog).

8c.D · Q2 — halftone dot pattern (LOCKED)

Baseline gradient + cream halftone dots @ 5%

Baseline 88/65/92 alphas at the desaturated #133d28 third stop (8c.B palette); layer a fine cream dot grid (8px, screen blend, 5% per dot) on top. The dots read as risograph / printed-programme texture — same vocabulary as the fanzine source mockups — without committing to a heavier opaque overlay. Iterated 0.18 → 0.10 → 0.05 alpha until the dots stopped competing with the photo.

Pro Adds retro print texture. Reusable as a tokenized pattern (--pattern-halftone-dots) for other dark surfaces.
Con New primitive; needs Storybook story + VR baseline. Pattern can moiré on smaller viewports.

Baseline gradient + barber-pole jersey stripes
