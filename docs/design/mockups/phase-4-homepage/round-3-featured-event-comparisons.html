Phase 4 · Round 3 · Featured event placement

Where does the next featured event live on the homepage?

Today the homepage fetches one event from EventRepository.findNextFeatured() (next upcoming event with featuredOnHome = true) and tucks it inside <NewsGrid> as a special card. Phase 4 redesigns that decision. Below: 4 placements, all using the same event fields (title, dateStart, optional dateEnd, coverImage, optional externalLink).

Locked from prior rounds. Hero shape = D.1 (carousel · strip-below thumbs). Hero source = S.2 (article.featured boolean). The schematics below show the body of / in compressed form so you can see the event's relative position.
