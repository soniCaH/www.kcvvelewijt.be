Phase 4 · Round 7 · SponsorsBlock

How are sponsors arranged on the homepage?

SponsorsBlock receives all sponsors with tier === "hoofdsponsor" or tier === "sponsor" (sympathisanten excluded per reference_sponsor_tiers). Realistic data: 1–3 hoofdsponsors and 8–15 sponsors. Logos are greyscale by default and reveal full colour on hover (CSS filter: grayscale(100%) + transition). This round picks the layout.

Locked from prior rounds. Section sits last in the body before SiteFooter (Spine A · BS.1 keeps bannerSlotC just before sponsors). Greyscale-default colour-on-hover is the locked logo treatment. This round picks the grid.
L.1
Tiered grid · main row + second row
Hoofdsponsors in a top row of 3 (larger logos). Below them, sponsors in a row of 5 (smaller logos). Hierarchy is explicit. Most conventional sponsor-page pattern.
Tradeoffs + Tier hierarchy crystal clear
+ Easy editor mental model
− Two distinct logo sizes per row
− Top row breaks if < 3 hoofdsponsors

L.2
Single equal grid · all sponsors equal size
No size hierarchy. All hoofdsponsors and sponsors render at the same logo size in a single 5-column grid. Order: hoofdsponsors first, then sponsors. Cleanest grid; loses the tier-emphasis cue.
Tradeoffs + Cleanest grid, single sizing rule
+ Most paper-newspaper register
− Tier hierarchy invisible
− Hoofdsponsors lose emphasis they paid for

L.3
Classifieds · irregular newspaper grid
12-column woven grid. One large hoofdsponsor (4 × 2 cells) anchors. Remaining hoofdsponsors take 4 × 1 cells. Sponsors fill 2-cell slots. Most fanzine / classifieds-page feel. Slot sizes hand-tuned per tier.
Tradeoffs + Most magazine-feeling register
+ Tier hierarchy via slot size, not row
− Slot-tuned for specific N values
− Lead hoofdsponsor gets disproportionate emphasis

L.4
Horizontal scroll-track · sponsor reel
Single-row horizontal scroll. Hoofdsponsors first (slightly larger), sponsors after. Compact vertical footprint. Reads like a sponsor-banner reel that you scroll through. Mobile-friendly by default.
Tradeoffs + Compact vertical (1 row)
+ Mobile-native (already scrolls)
− Hides ~half the sponsors below the fold
− No grid presence on the page
