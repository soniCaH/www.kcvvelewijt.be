Locked spec — primitive composition (no new vocabulary)

  • Shell → <TapedCard> cream, border-2 border-ink + hard offset shadow + one <TapeStrip> (warm, top-left). Sharp corners.
  • Kicker → <MonoLabel variant="plain"> (jersey-deep raw label-token span).
  • Headline → <EditorialHeading level=1> Freight upright; optional one-word italic jersey-deep accent; warm "." terminator. Scales up in the no-image (typographic) state.
  • Lead → italic font-display, auto-hides when absent.
  • Image → <TapedFigure aspect="landscape-4-3"> newsprint (colour), single tape strip; desktop right column, mobile below (m1). Absent → typographic (n1): no slot, headline owns the card, <DottedDivider> for texture.
  • CTA → optional <LinkButton> slot (parity with legacy; none of the 4 pass it today).
  • size → "default" | "compact" (compact: tighter padding, smaller headline, image suppressed) for loading skeletons + utility pages.
