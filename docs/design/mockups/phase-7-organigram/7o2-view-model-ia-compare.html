Decision 7o2 — pick the view model

C reskins today's 3 tabs (solves nothing structurally). B collapses to Structuur⟷Hulp (kills redundancy, still mode-switch). A goes full single-scroll like the rest of Phase 7 (best consistency, but an inline interactive diagram fights scroll → forces a tame 7o3 + no 3-D home). D ★ keeps the calm single-scroll for browse+Hulp but makes the diagram an opt-in fullscreen verkenner — the only option that fixes all four pains and gives the literal 3-D idea a place to live.

Whatever wins sets the container for 7o3. A → inline tame tree. B → diagram as a sub-render. D → immersive focused mode (pan-zoom / drill / 3-D all viable there). A “D but with a small inline structure preview that expands to fullscreen” hybrid is available on request.
