Phase 5 · Round 1 · Drill #1785 · 5.d3

Section-break flourish — diamond / seam / dotted / none

Drill resolves the mid-article section-break visual treatment used by <QASectionDivider> (major Q&A section breaks) and any future article body <h2> renderer. The 5d3 lock determines whether <QASectionDivider flourish="diamond"> joins the existing flourish="em-dash" | "star" API or another option wins.

Each option renders the same article fixture (three sections, two breaks) at --container-prose width (680px) on cream — same scale, same copy, so the rhythm difference is visible. Each break carries a section subtitle ("Het begin." / "Het seizoen.") to test how the flourish composes with text. The first paragraph after every break drops back to standard serif body — no drop-cap (those only land at the article opener, locked elsewhere).

Option A · Diamond glyph
Single ◆ glyph + thin-rule subtitle
Diamond glyph (◆, Unicode U+25C6) in jersey-deep, centered above the section subtitle which keeps the existing flanking-thin-rules treatment. Typographic flourish, no new asset.
Vocabulary ✓ Existing thin-rule subtitle (Phase 3-b locked)
✓ --color-jersey-deep
Δ Net-new flourish="diamond" on <QASectionDivider> API

"Halfweg de eerste helft, derde wedstrijd. Een loopactie waarvan ik wist: hij snapt het. Pas dan wist ik dat dit ging lukken, dat hij hier hoorde."

Lars knikt. "Wim heeft dat dan ook gezegd, achteraf. In de kleedkamer. Niet hardop, gewoon zo'n knik. Dat is wel groot, hè."

◆
Het seizoen.

Het seizoen begon nochtans niet zoals iedereen had verwacht. Drie wedstrijden zonder zege, een keeperswissel, en een vergadering die langer duurde dan het wedstrijdverslag.

"Dat was ook gewoon ongelukkig," zegt Lars achteraf. "We waren er nog niet, en dat kon je ook zien."

◆
Wat als je nu kon kiezen?

"Hetzelfde, denk ik," zegt Lars. "Misschien niet die warming-up vóór Putte. Maar voor de rest? Hetzelfde."
