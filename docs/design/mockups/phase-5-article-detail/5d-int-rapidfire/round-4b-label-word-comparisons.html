Rapid-fire — label word (Dutch alternatives to "Sneltrein")

Round 4 locked the opener treatment (MonoLabel between hairlines). This round picks the actual word. Everything else is held constant — only the label text changes between the hairlines.

"Sneltrein" was the Phase 2 word; user has flagged it as not the right register for the redesign. Four Dutch alternatives below. Casing follows the MonoLabel convention (all-caps display, mixed- case authored value).

Option A · Kort & krachtig
Klassieke Nederlandse interviewtaal
Standard Dutch interview vocabulary — "short & punchy". Most journalistically conventional. Reads as a familiar register; signals "fast Q&A" without effort. Two words + ampersand may look busy at MonoLabel scale.
Register ✓ journalistic / interview-genre familiar
✓ neutral & safe
⚠ 3-token label (slightly long)

Option B · Recht voor de raap

Option C · Snelle ronde
Sport-neutraal
"Quick round" — neutral, sport-adjacent. No metaphor, no idiom; describes what's on the page. Shortest of the four. Closest translation of "rapid-fire round" without train imagery. Risks blandness — could just as easily be in any club's match report.

✓ neutral / unambiguous

Editorial vs. authored
  • This label is not editor-authored per qaBlock; it's a fixed string the renderer emits whenever a rapid-fire group is detected. So the label is part of the design lock, not a Studio field.
  • If we want editor override later, a QaSection-level label field could be added (out of scope for this drill). For now: one canonical word, locked here.
Other candidates considered
  • Snel & kort — too similar to "Kort & krachtig"; less idiomatic.
  • In het kort — reads as "in brief" / a summary, wrong meaning.
  • Vragenronde — generic round-of-questions; doesn't carry the rapid feel.
  • Tot slot — wrong meaning (implies an ending).
