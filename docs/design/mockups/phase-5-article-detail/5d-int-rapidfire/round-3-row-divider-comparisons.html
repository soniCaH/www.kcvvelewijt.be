Reuse signal
  • Option A (dotted ink-muted) is the only option that explicitly reuses the locked standard QARow divider — rapid-fire reads as the same family as standard, just compressed. Per feedback_reuse_approved_primitives, this is the path of least new vocabulary.
  • Options B and D introduce vocabulary that doesn't appear anywhere else in the redesign (hairline solid, mono dot ornament). Worth weighing the visual gain against that drift.
  • Option C goes the opposite direction: no chrome at all, banking on pure rhythm. Cleanest if rapid-fire answers stay genuinely short.
