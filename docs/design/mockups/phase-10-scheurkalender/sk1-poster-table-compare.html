What's fixed either way

  • Data: full-season A + B league fixtures (played + upcoming), via bff.getMatches(aTeamId) + getMatches(bTeamId), filtered to league only (competitionType === "league" — the competition field is a division name like "3de Nationale", not "Competitie"), A/B labelled by which team was queried, merged + sorted by date.
  • Columns: date (za/zo DD/MM[/YY]) · time · home · away. KCVV bolded with the A/B inline in the name. No logos, no pills, no scores.
  • Private: robots: noindex, no nav/footer/sitemap link — reachable only by typing the URL.

The difference

  • A print-clean — white, Montserrat, hairline rules, a heavier rule splitting each weekend. Pastes onto the textured poster bg with zero brand colour fighting it. Closest to what you screenshot today.
  • B poster-matched — cream, a per-weekend mono label, Freight masthead, faint jersey-deep tint on the rows where KCVV is home. More "on-brand" on screen, but the tint/labels are extra marks you'd be cropping around in InDesign.
