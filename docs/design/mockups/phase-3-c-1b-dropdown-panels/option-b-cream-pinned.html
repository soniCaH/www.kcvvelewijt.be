Sectie · De club

Een club met een eigen plek aan de Hollestraat

KCVV Elewijt speelt al meer dan honderd jaar in de tweede helft van het amateurvoetbal. Onder de eik aan de Hollestraat is de zaterdagochtend een gewoonte — kinderen, ouders, koffie, gevolgd door een halve liter en een match.

Reads as: a flyer pinned beneath the masthead. The 2px border + hard offset shadow are the same vocabulary <TapedCard> and editorial chrome already use elsewhere — most "fanzine" of the four. The 6px gap between header seam and panel edge announces "this is a separate sheet of paper."

Tokens used

  • --color-cream-soft
  • --color-ink (2px border)
  • --shadow-paper-sm (hard)
  • --color-ink (text)
  • --color-ink-muted (em-dash bullet)
  • --color-jersey-deep (hover)

Pros

  • Strongest fanzine register — paper-shadow vocabulary is the redesign's signature.
  • Heavy 2px ink border separates panel cleanly from header at any scroll position.
  • Hard offset shadow signals "this is a card-like artefact" — feels like a poster pinned to a noticeboard.
  • Distinct from chrome Surface — never confused with a system menu.

Cons

  • Hard offset shadow ignores master design §3.6 advisory ("blurred shadows for chrome").
  • 2px border + offset shadow against scrolling page content can feel heavy on long pages.
  • 11-item De club panel becomes a tall heavy slab — risks visual weight imbalance.
  • Implementation: requires hover-state arbitration so the shadow doesn't collide with the next nav row's content.

What changes if Q1 locks B

All three submenus (Teams, Jeugd, De club) get the cream-soft + 2px border + hard offset shadow treatment. Sticky-scroll behaviour gains weight — the page content scrolling under the panel will pass behind a 4px hard shadow band. Q3 Composition is unchanged.
