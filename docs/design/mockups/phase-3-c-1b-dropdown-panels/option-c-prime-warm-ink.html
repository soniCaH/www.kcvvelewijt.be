Sectie · De club

Een club met een eigen plek aan de Hollestraat

KCVV Elewijt speelt al meer dan honderd jaar in de tweede helft van het amateurvoetbal. Onder de eik aan de Hollestraat is de zaterdagochtend een gewoonte — kinderen, ouders, koffie, gevolgd door een halve liter en een match.

Reads as: inverted newsprint. The warm sepia-tinted dark sits on a cream page like a black-ink-on-old-paper print object — same inversion as C, but the warmth keeps the panel inside the redesign's "printed warmth" register instead of dropping out into "system chrome". The cream-soft caps text reads as old printer ink rather than UI metal.

Proposed new token: --color-ink-warm: #1d1812 (plus warm-tinted hairline + bullet companions). If C′ wins Q1, these get added to the redesign palette in apps/web/src/app/globals.css as new tokens. They are panel-only — body text stays on --color-ink.

Tokens used (proposed)

  • --color-ink-warm (bg, new)
  • --color-ink-warm-rule (1px border, new)
  • --shadow-paper-sm-soft
  • --color-cream-soft (text)
  • --color-ink-warm-bullet (em-dash, new)
  • --color-jersey-bright (hover bullet)
  • --color-cream (hover text)

Pros

  • Same legibility win as C — dark panel, cream caps, impossible to miss.
  • Warmth keeps the panel inside the cream paper world rather than feeling like a system menu.
  • Reads as "printed inverse" — the redesign's "printed, handled" register applied to chrome.
  • Hover state can use jersey-bright safely — warm dark provides better visual context for the bright accent.

Cons

  • Adds 3 new colour tokens to the palette (small but real).
  • Jersey-deep on warm-ink still fails AA — must use jersey-bright (same constraint as C). Unless we say warm panels = "dark surface mode" with carved-out exceptions for the green.
  • Warm-ink is a printed-newspaper register — could feel mannered if used outside dropdowns.
  • Slightly less visually decisive than full ink-soft — the warmth softens the contrast jolt that some might want.

What changes if Q1 locks C′

The redesign palette gains a "warm-ink" sub-family scoped to inverted chrome surfaces (dropdowns now; potentially modals or command palette later). The --color-jersey-bright carve-out is formalised as the dark-surface accent. Composition Q3 should consider whether the cream-soft text needs a slight weight or tracking shift to hold up at 11px on warm-dark — usually fine but warrants a check.
