Icon library comparison.

Phase 2 design checkpoint — visual A/B for the icon strategy. See docs/plans/2026-04-27-redesign-master-design.md §9 open question 8 (resolved 2026-04-28 → see decision-log entry 17). Lucide is what the site uses today; the redesign direction (retro-terrace fanzine) wants something that sits with cream paper, taped polaroids, and Freight Display italics rather than fighting them.

Open this file in a browser. Icons load from public CDNs (jsdelivr); some may take a moment on first paint. If a square stays blank, the CDN doesn't have that variant — note it and move on; absence in a CDN doesn't necessarily mean the library lacks the icon.

How to read this

  • Lucide column: the current site icons. Reference baseline. Geometric, modern, 2px stroke, perfectly aligned.
  • Phosphor (Fill) column: same shape language as Lucide but solid ink-stamp weight. Cheapest swap from Lucide.
  • Game Icons column: hand-drawn engraving feel. Free, CC BY 3.0. SVGs ship white-on-transparent so they're CSS-inverted here for visibility on cream.
  • Typographic Glyph column: a Unicode character typed in body or mono font. Zero icon-library payload.
  • Custom KCVV (AI-authored) column: SVGs authored on-demand by Claude during component implementation. Single style guide — 24×24 viewBox, solid currentColor fill, bold silhouettes with slight irregularity, no library dependency. Samples below are first-draft to give you a feel; iterate at implementation time. Strongest brand fidelity, no "missing icon" problem, ~5–15 min per icon to author.

Reading the comparison: Phosphor-fill is the safest zero-friction swap. Custom KCVV (AI-authored) gets you bespoke ink-stamp glyphs that read on-brand for the fanzine direction, at the cost of ~5–15 min of authoring per icon. Where the typographic glyph reads cleanly, that's still the cheapest answer (no SVG at all). Game Icons is rejected — its engraving style fights the cream/ink palette in practice.

The 7 custom samples shown are first drafts. Real authoring at implementation time would tighten paths, add tiny irregularity for the hand-drawn feel, and commit to a single line-weight or solid-fill rule across the whole set. The point of these samples is to test whether the AI-authored direction is visually credible — not to be production-final.

Final decision lands at the Phase 2 atom-rework design checkpoint.
