# Colors

All color values live in `src/app/globals.css` inside `@theme {}`.
The swatches below read directly from the live CSS custom properties — no hard-coded duplicates.

The legacy `--color-kcvv-*`, `--color-green--*` digital palette was retired in
the redesign (#2154). The site now ships a single retro cream-on-ink surface
vocabulary — the tokens below are the canonical set.

## Usage in code

```tsx
// Tailwind utility classes (most common)
<div className="bg-cream text-ink">…</div>
<button className="bg-jersey-deep text-cream hover:bg-jersey">…</button>

// CSS custom property (when Tailwind class not available)
<div style={{ color: 'var(--color-ink-muted)' }}>…</div>
```
