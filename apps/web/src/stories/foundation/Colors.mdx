## Usage in code

```tsx
// Tailwind utility classes (most common)
<div className="bg-kcvv-green text-kcvv-gray-blue">…</div>
<button className="bg-kcvv-green hover:bg-kcvv-green-hover">…</button>

// CSS custom property (when Tailwind class not available)
<div style={{ color: 'var(--color-kcvv-gray-dark)' }}>…</div>
```

## Redesign / Phase 0 — Cream Surface

The redesign introduces a cream-on-ink surface palette alongside the existing
brand greens. Both vocabularies coexist — legacy `--color-kcvv-*` tokens remain
valid until the Phase 9 cleanup, so feature work landing during Phases 1–8 may
read either set without aliasing. Pick the cream surface for new redesign
primitives; keep `--color-kcvv-*` for components that have not yet been
restyled.

Each row below renders the live token via `var(--color-...)` so the swatch and
the contrast notes always reflect what `globals.css` ships today. Contrast
ratios are measured against `--color-cream` (`#f5f1e6`).
